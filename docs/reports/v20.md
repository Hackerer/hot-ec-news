## 本轮自问自答

问题：

- Markdown 改好了，为什么还要立刻改邮件？

回答：

- 因为你最终接收的很可能就是邮件。日报和邮件如果还是两套结构，需求方验收时会直接认为结果不一致。

## 本轮目标

- 让 Email HTML 跟 Markdown 使用同一阅读顺序
- 每个类目先展示整体 Top15
- 每个类目再展示平台 Top15

## 交付内容

- `renderEmailHtml` 现在会输出：
  - 类目标题
  - `整体搜索热词 Top15`
  - 平台 Top15 子区块
- 平台子区块同步展示：
  - 第一/第二信源
  - 该平台在该类目下的总词数
