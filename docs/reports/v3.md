# v3 报告

日期：2026-03-14

## 本轮自问自答

问题：

- 第三方平台没有稳定公开 API 时，怎么做出可用的第二信源校验？

回答：

- 先支持后台导出的 CSV 文件导入，再在聚合层做可信度计算。这比直接硬抓第三方网页更稳定。

## 本轮目标

- 支持第三方 CSV 导入
- 支持蝉妈妈/飞瓜这类导出格式
- 增加可信度和校验状态
- 生成带“第三方校验结果”的日报

## 交付内容

- `importThirdPartyCsv`
- `import:third-party` 流程
- `buildValidatedReport`
- 校验状态：`validated / primary_only / secondary_only`
- 置信度计算
