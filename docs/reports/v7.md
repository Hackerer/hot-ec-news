# v7 报告

日期：2026-03-14

## 本轮自问自答

问题：

- 如果日报只有“今天榜单”，那每天看起来都像一次性的快照，业务价值够吗？

回答：

- 不够。日报必须带历史对比，至少要知道哪些词是新上榜，哪些词是连续上榜且变化更快。

## 本轮目标

- 给日报增加历史对比能力
- 增加新增爆发词
- 增加连续上榜词
- 在类目列表里展示较上次变化

## 交付内容

- 趋势状态：`new / up / down / steady`
- 报告区块：`新增爆发词`
- 报告区块：`连续上榜词`
- 类目行展示 `较上次 +/-delta`
- 数据库支持读取上一采集日
